# Homebrew Tap for Arroxy

Install Arroxy on macOS via [Homebrew](https://brew.sh):

```
brew tap antonio-orionus/arroxy
brew install --cask arroxy
```

The cask in this tap is auto-bumped on each [Arroxy](https://github.com/antonio-orionus/Arroxy) release by GitHub Actions.

> The macOS DMG is unsigned. On first launch macOS will block it; right-click → Open, or follow the workaround in the [main README](https://github.com/antonio-orionus/Arroxy#first-time-launch-on-macos).
